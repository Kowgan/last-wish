Last Wish

An open source Ultima Online RunUO Freeshard hoping to fulfill one last wish of UO Players.
Every one is welcome!

Three Steps to Join

  1. Download and install UO Client 7.0.15.1.
  2. Download and install Razor
  3. Launch Razor and seutup: Server: lastwish.servegame.com | Port: 2593

Source Code

You can download the shard source code here, and run your own custom Last Wish server!
