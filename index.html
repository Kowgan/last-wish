Last Wish

An open source Ultima Online RunUO Freeshard hoping to fulfill one last wish. Everyone is welcome!

Three Steps to Join

  1. Download UO Client 7.0.15.1 (Installer or Portable)
  2. Download and install UOS
  3. Launch UOS and configure like this:


Server: lastwish.servegame.com
Port: 2593

Shard Activity

Source Code

You can download the shard source code here, and run your own custom Last Wish server!

Player Commands

To use a command, just type it on the game preceded by "[". For example: [Look

A Usage: A <text>
Broadcasts a message to all online guildmembers and allies.
AddToParty Usage: AddToParty <Name>
Used to add a player to your party. Optionally you can provide a name to filter.
AFK Usage: AFK [<message>]
Puts your char in 'Away From Keyboard' mode.
AutoDefend Usage: AutoDefend
Enables or disables the auto-defend feature.
AutoSheathe Usage: AutoSheathe
Enables or disables the weapon auto-sheathe feature.
BoatControl Usage: BoatControl
Shows a menu to control a boat.
C Usage: C <message>
Broadcasts a message to all players online. If no message is provided, it opens the Chat History.
ChangeCharacter Usage: ChangeCharacter
Changes your character without needing to logout.
ChatHistory Usage: ChatHistory
Opens the Chat History.
ChatToggle Usage: ChatToggle
Enables or Disables the Chat.
Convoke Usage: Convoke [filter]
Convokes a player to join you.
Dismantle Usage: Dismantle
Dismantles an addon from your house.
Dump Usage: Dump
Allows you to dump everything from one container to another.
E Usage: E <sound>
Emote with sounds, words, and possibly an animation with one command!
Equip Usage: Equip
Equip a set of items from a container.
Event Usage: Event
Used to join shard events when they're open.
Forum Usage: Forum
Opens the forum.
G Usage: G <text>
Broadcasts a message to all online guildmembers.
Help Usage: Help [<command>]
Gives information on a specified command, or when no argument specified, displays a gump containing all commands
Issue Usage: Issue
Alias: Bug
Views a list of the current issues.
Look Usage: Look
Used to look at someone who is near you or to change your's character description.
MoveItems Usage: MoveItems
Allows you to move items of a specific type from one container to another. For example, you want to move all spools of thread from your main pack to another backpack. Type [moveitems, target a spool of thread, then target the backpack you want the spools of thread to be moved to. The TO container normally is outside your main backpack. If you type ' exact' after the command [moveitems it will only move very specific types. For example, [moveitems and targeting a yumi bow will move ALL baseranged items. [moveitems exact and targetting a yumi will only move yumis.
MyHouses Usage: MyHouses
Displays a menu listing all your houses. The menu also contains specific house details, and options to: go to house, and demolish house.
Notoriety Usage: Notoriety
Tells your fame and karma.
Password Usage: Password
Changes your account's password.
PM Usage: PM [filter]
Send a private message to any online player.
Pray Usage: Pray
Resurrects you if you have positive karma.
Recall Usage: Recall
Takes you back to where it all began.
Sin Usage: Sin
Resurrects you if you have negative karma.
Site Usage: Site
Opens the shard website in your browser.
Statistics Usage: Statistics
Shows shard statistics.
Status Usage: Status
Show the server status, including a list of all players online
Time Usage: Time
Returns the game's time and the server's local time.
Where Usage: Where
Tells the commanding player his coordinates, region, and facet.
